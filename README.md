# Backyard Putting Tracker

A simple, family-friendly web app for tracking putting rounds in the backyard.  
Built as a personal project to track scores, view history, analyze players, and store data using Firebase.

## ğŸŒï¸ About the Project

The Backyard Putting Tracker allows you to:

- Start putting rounds with customizable players  
- Score 20 holes hole-by-hole  
- Track full round history  
- View analytics such as:
  - Player handicap trend
  - Best rounds
  - Course difficulty (average score per hole)  
- Export your data to CSV or JSON  
- Sync data using Firebase (optional)

All logic is contained in `index.html`, making it easy to deploy on GitHub Pages.
